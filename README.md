# [NightCat](http://nightcat.win/)

------

## 目的
通过建站来深入学习各种技术，并搭建一个能展示自己的平台。

## 技术栈
前端：React + Redux + Webpack + HTML5/CSS3(游戏)<br />
后端：NodeJS + Express + MongoDb<br />
后台管理系统： Hbs + Jquery + Gulp <br />
